# Jogos Digitais e Aprendizado Ativo: uma abordagem para o ensino de lógica de programação

Repositório com materiais utilizados em uma pesquisa sobre o uso do Minecraft Education no ensino introdutório de programação, com foco em metodologias de aprendizagem ativa.

O repositório inclui:

- listas de atividades;
- slides utilizados em aula;
- formulários aplicados;
- scripts de análise;
- materiais complementares utilizados na pesquisa.

## Estrutura do repositório
```
.
├── analises/
├── formularios/
├── materiais-de-aula/
├── listas-de-atividades/
└── README.md
```

### Organização dos materiais
`listas-de-atividades/`

Contém as listas de atividades utilizadas durante as aulas.

`materiais-de-aula/`

Contém slides e outros materiais de apoio utilizados nas aulas.

`formularios/`

Contém os formulários aplicados antes e após as atividades.

`analises/`

Contém scripts e notebooks Python utilizados na análise dos dados da pesquisa.
